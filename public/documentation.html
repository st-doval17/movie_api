Welcome to myFlix API.

This is a web application that will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Allow new users to register /users POST A JSON object holding data about a new user to add, e.g., 
  {
    "name": "Mary",
    "favoriteMovies": []
  }
A JSON object holding data about a new user to add, with ID generated e.g., 
          {
            "name": "Mary",
            "favoriteMovies": [],
            "id": "90b014b6-88bf-480b-9e35-8cfae613de76"
          }
Allow users to update their user info (username) /users/:id PUT none A text confirming username has been updated e.g., 
          'Successful. Your username has been updated'
Allow users to add a movie to their list of favorites /users/:id/:title POST A JSON object holding data about a user e.g., 
  { 
    id: 1, 
    name: "Andie",
    favoriteMovies: []
  }
A text that a movie has been added e.g., 
          'Casablanca has been added to user 1's array'
Allow users to remove a movie from their list of favorites /users/username/movies/:title DELETE none
A text that a movie has been removed e.g., 
          'Casablanca has been removed from user 1's array'
Allow existing users to deregister /users/:id DELETE none A text that a user email has been removed e.g., 
          'user 1 has been deleted'
